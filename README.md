# Student Manager API

A REST API built with FastAPI and SQLite for managing student records.

## Endpoints
- POST /addstudent — Add a new student
- GET /readstudents — Get all students
- GET /students/{id} — Get student by ID
- PATCH /updatestudent/{id} — Update a student
- DELETE /deletestudent/{id} — Delete a student

## Tech Stack
- Python
- FastAPI
- SQLite
- Uvicorn
